# cycle_engine
This is a game engine I've been working for a few months now. 

So far, its renderer features:
  - deferred rendering
  - shadow mapping
  - normal mapping
  - parallax occlusion mapping
  - bloom
  - volumetric lights
  - PBR materials
  
Translucent materials are on the way.  

Physics engine used:
  - Bullet physics
  
Sound playback (deactivated):
  - OpenAL
